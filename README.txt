Mi pre proyecto "PP_Coder" funciona en base a dos aplicaciones, app_views que gestiona las vistas estáticas y app_data que gestiona base de datos y formularios.

Cada aplicación funciona de manera independiente de la otra y gestiona su propia herencia html.

Su interfaz es bastante intuitiva, cuenta con tres formularios independientes para insertar registos a las respectivas tablas en la base de datos. 

Cuenta además, con la posibilidad de visualizar todos los registros añadidos a cada tabla con sus correspondientes opciones de filtrado, este puede ser por cualquiera de las variables de los registros, texto, números, fechas, pudiendo dejarse el campo de filtrado en blanco si se quiere acceder a todos los datos almacenados.
